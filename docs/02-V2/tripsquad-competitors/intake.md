# Intake: TripSquad

*Skill: startup-competitors | Generated: 2026-05-21*

## Product

TripSquad is an iOS-first group trip planning app that replaces WhatsApp + Google Docs + Splitwise for group travel.

## Core Features

- City pin drop with places feed
- Shared shortlist with voting
- Day-by-day timeline
- Photo sync / auto-tagging
- Public shareable trip pages

## Business Model

Organizer-pays subscription: $39/yr. Group members join free.

## Known Competitors

Wanderlog, TripIt, Polarsteps, Tricount/Splitwise, Google Maps trip features, Apple Maps guides, TripAdvisor trip planning, Sygic Travel.

## Market

Group travel planning (iOS-first, targeting friend/family group organizers).
